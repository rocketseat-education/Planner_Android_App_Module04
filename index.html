The Planner application ("we", "our" or "application") respects and protects the privacy of its users. This Privacy Policy describes the practices adopted by the application in the processing of personal data, in accordance with Google Play guidelines and applicable best practices for data protection.

1. Scope and Acceptance

By using the Planner application, the user agrees to the terms of this Privacy Policy. If the user does not agree with the terms described herein, it is recommended that they do not use the application.

2. Information Collected

Planner collects only personal information that is voluntarily provided by the user and strictly necessary for the operation of the application:

  • Name;
  • Phone;
  • Email address;
  • Data related to tasks (title, date, time and status).

All information is stored exclusively on the user's device. The application does not perform automatic collection, does not communicate with remote servers and does not share data with third parties.

3. Purpose of Processing

The stored data is intended solely and exclusively to provide the core functionalities of the application, such as:

  • Registration, editing and deletion of tasks;
  • Organization by date and time;
  • Marking tasks as completed or not completed;
  • Optional display of user identification data.

No data is used for marketing, advertising, profiling or external analysis purposes.

4. Storage and Security

Data is kept in the device's internal database, without transmission to external servers, APIs or third-party services. The application adopts good development practices for the protection of information, although final security also depends on measures adopted by the operating system and by the user themselves (e.g.: screen lock).

5. Data Sharing

Planner does not share, sell, rent or transfer any personal data to third parties. There is no integration with third-party services for information exchange.

6. Permissions

The application does not request sensitive permissions during execution. Planner does not access camera, microphone, contacts, location, external storage, notifications or other resources considered sensitive by the operating system.

7. User Rights and Data Control

The user has full control over the data entered in the application. It is possible to edit or delete information at any time in the application interface. Uninstalling the application permanently removes locally stored data, without external copies.

8. Retention

Data remains on the device while the application is installed. Automatic cloud backups are not performed by the application.
